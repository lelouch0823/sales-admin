# 3. 信息架构（菜单与页面）

建议左侧导航结构：

1. **Dashboard**（关键指标）
2. **今日推荐**
   - 全局推荐
   - 门店推荐
   - 推荐预览
3. **商品管理 (PIM)**
   - 商品列表
   - 新建商品
   - 导入任务中心
   - 媒体资源管理
4. **库存中心**
   - 库存总览
   - 库存查询 (Stock Explorer)
   - 入库/出库单
   - 调拨单
   - 预订管理
5. **客户管理**
   - 客户列表
   - 公海客户池
   - 客户共享（成员/权限）
6. **用户与权限**
   - 用户管理
   - 角色管理
7. **审计日志**
   - 操作日志列表与详情

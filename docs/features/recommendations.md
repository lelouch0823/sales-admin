# 4.1 今日推荐（手动推荐）模块

#### 4.1.1 业务规则
- 推荐位仅：**今日推荐（home_today）**。
- 支持两级配置：
  - **全局（tenant_id = NULL）**：总部统一推荐
  - **门店（tenant_id = X）**：门店推荐 **完全覆盖**（门店存在配置时，不用全局补齐）。
- 存在**可预订/可调货**：缺货不一定不可售。

#### 4.1.2 全局推荐页面
- 列表展示字段：主图、SKU、名称、状态、生效时间、优先级。
- 操作：新增商品、排序、启停、删除。
- 校验：需校验商品必须为可售（未下架）。

#### 4.1.3 门店推荐页面
- 顶部：门店选择器。
- 列表同全局，增加本门店库存状态展示。
- 新增时强提示：
  - 若 `UNAVAILABLE`：禁止添加。
  - 若 `BACKORDER`：显示提示。
- **完全覆盖说明**：“本门店配置存在时，App 今日推荐仅展示本门店配置，不会用全局补齐。”

#### 4.1.4 推荐预览页面
- 选择门店 + limit，展示“App 端最终看到的推荐列表”。
- 明确显示 source (global/store) 和库存状态。

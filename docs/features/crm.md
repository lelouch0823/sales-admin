# 4.2 客户管理模块（公海 + 共享）

#### 4.2.1 客户列表页面
- 筛选：scope（我的/公海/共享/全部）、标签、搜索。
- 列表字段：姓名、状态标识、跟进时间。
- 操作：详情、编辑、变更负责人、移入公海/领取。

#### 4.2.2 公海客户池页面
- 仅展示 `owner_user_id = NULL` 的客户。
- 支持：领取、批量分配、回收。

#### 4.2.3 客户详情
- 分区显示：基本信息、标签、跟进记录。
- 操作：管理共享成员。

#### 4.2.4 共享客户（协作成员）管理
- 提供“共享成员”管理。
- 成员列表：user、role（viewer/editor/manager）。
- 权限规则：owner 或 manager 可管理共享。
